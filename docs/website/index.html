— What's improved over JTDX

No more dropped replies.
No more endless CQ loops.

WKjTX fixes the JTDX auto-sequencer and adds the tools every HF DX operator needs — all built in, no companion app required.

01
Reliable CQ Replies

No longer drops replies in marginal copy. If a station answers your CQ and the decode is valid, WKjTX responds — even on a noisy band.

02
No Stuck CQ Loops

The sequencer stops calling CQ when nobody is coming back, instead of cycling indefinitely. You stay in control of your transmit time.

03
JTDX Decoder Quality

Built on JTDX 2.2 — the iterative multi-pass FT8 decoder that routinely out-decodes stock WSJT-X on crowded HF bands. +5–15% more decodes in typical conditions.

★

3-radio quick-switch — one click, one rig

NEW in v1.1. Three preset slots in the menubar corner. Switch radios without ever opening Settings.

Top-right corner of the menubar · active slot in amber · empty slots show "+"

Run more than one rig from the same PC? Click a slot, the radio switches — CAT, audio device, PTT, the lot. No Settings dialog. No quitting and re-launching. No companion app.

  • Slot 1 — your base configuration. Always there, always safe. Editing it goes through normal Settings.
  • Slots 2 & 3 — independent overlays. Each lives in its own INI file under %LOCALAPPDATA%/WKjTX/profiles/. They never touch your main config.
  • Left-click a named slot → instant switch (CAT reconnect, audio re-init, all in-app).
  • Left-click "+" → opens the compact configurator: rig dropdown, CAT & PTT serial port pickers, baud, handshake, audio in/out devices.
  • Right-click any slot → Configure / Rename / Hide / Clear.
Why this matters
  • Mobile + base setup — IC-705 in the field, IC-7300 at home, swap in seconds.
  • SO2R-style operation — second radio on a different sound card, ready to go on the same WKjTX install.
  • Test rig swaps — try a borrowed radio without losing your base config.
  • Zero risk — Slot 1 is read-only from the profile button. Mistakes in slot 2/3 stay there.
!

Auto-call — hands-free DXing

Seven trigger categories. Five alert slots. Hardcoded rate limits so you never flood the band.

⚠ Use responsibly. With auto-call ON the station transmits unattended. Never leave the station unattended with auto-call ON while connected to a linear amplifier or a timer-based power strip. You are always responsible for whatever goes on air under your licence.

Open File → Auto-call and WKjTX starts replying to decoded calls automatically — based on the categories you enable. Works in the background while you do other things.

  • NEW DXCC / NEW CQ zone / NEW ITU zone — the DX categories you can't afford to miss during an opening.
  • NEW grid 4-char / NEW prefix / NEW callsign — for the chasers who want every new one.
  • Alert callsigns — 5 dedicated slots. Perfect for "5T0SP called, I was in the kitchen".
Built-in safeguards (not user-editable)
  • 120 s cool-down per callsign — never called twice in a row
  • 3 auto-calls / minute global rate cap
  • Pulsing badge in status bar — impossible to miss it's on
  • All OFF by default — opt-in per category
File → Auto-call: seven opt-in categories · all OFF by default

All reference data — one button

Hit Update data once on first run. Four files downloaded, verified, installed. Every URL is editable.

Settings → General → Data updates · last update timestamp shown

No more hunting down four separate files from four websites. One button fetches everything — and shows you when it last ran and whether all files are current.

  • cty.dat — country / prefix data (Club Log)
  • state_data.bin — US state lookups
  • grid_data.bin — Maidenhead grid geometry
  • lotw-user-activity.csv — LoTW registered users
  • Editable URLs — change any field to point at a mirror; clear to restore default
  • Update Hamlib — opens the JTDX SourceForge page; drop in the new libhamlib-5.dll

Every feature you need

Built in. No companion app. No subscription.

01
ADIF Log Import

Import a .adi file from WSJT-X, JTDX, Log4OM or any other logger. Duplicates skipped on CALL + QSO_DATE + BAND + MODE.

02
ADIF Log Export

Save a snapshot of the internal log to any .adi file of your choice — without leaving the app. One click in Settings → General.

03
Font Size Spinboxes

Separate point-size spinboxes next to Application Font and Decoded Text Font. Adjust to taste with one click.

04
English · User Translations

Ships English only. Drop your compiled wkjtx_<locale>.qm in bin/translations/ to run in any language. Legacy jtdx_*.qm files work as-is.

05
Hamlib Updater

Opens the JTDX Hamlib SourceForge directory in your browser. Download, drop libhamlib-5.dll into bin\, done. Old DLL auto-renamed as backup.

06
Portable · Zero Install

Unzip anywhere — Desktop, USB stick, C:\WKjTX. Everything bundled: DLLs, decoder, Hamlib. Uninstall by deleting the folder.
