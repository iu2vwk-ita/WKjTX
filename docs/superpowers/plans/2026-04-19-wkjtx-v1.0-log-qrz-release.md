> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close out WKjTX v1.0: wire per-profile log routing so each profile can direct its QSOs to a shared or dedicated ADIF file, add qrz.com Logbook API upload (verify/extend if JTDX already has it), polish the NSIS installer, publish release.

Hook `QrzUploader` alongside the existing `eqsl.cpp` upload. Add credentials to the per-profile INI under `[Reporting/Qrz]` and `[Reporting/Log]` sections.

**Tech Stack:** Qt 5.15 `QNetworkAccessManager` for HTTPS to qrz.com.

**Prerequisites:** Plan 3 (v0.3.0) complete.
