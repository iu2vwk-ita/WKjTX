# WKjTX

A freeware fork of [JTDX](https://sourceforge.net/p/jtdx/) focused on multi-radio workflow,
stability, and auto-call safety. Built on the weak-signal decoder heritage of WSJT-X (K1JT)
and JTDX (HF community).

**Status**: design phase — no code yet. See `docs/superpowers/specs/` for the active design.
